# Kanemori

Personal finance tracker optimized for local-first stability and ease of use.

## Quick Start
- Open **Kanemori** from your desktop to start the app.
- Use **Kanemori Control** to manage the local server.

## Maintenance
See [GEMINI.md](./GEMINI.md) for detailed maintenance instructions and reliability features.

## Tech Stack
- **Framework:** Next.js 16 (App Router)
- **Database:** SQLite (better-sqlite3)
- **Styling:** Tailwind CSS
- **Visualization:** Recharts
